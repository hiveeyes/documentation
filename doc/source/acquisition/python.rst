*******
Library
*******
There **isn't** a convenient Python library for transmitting
measurement values using Python ready for download **yet**.

In the meanwhile, please have a look at the implementation prototype `backend_test.py`_
and some documentation about it at `backend.rst`_. Both are part of a `pull request`_
for a beehive monitoring project from Bremen, Germany based on RaspberryPi hardware.
